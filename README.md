## 🧮 Qo'llab-quvvatlanadigan arifmetik amallar

| Amal | Belgisi | Misol |
|------|---------|-------|
| Qo'shish | `+` | `5 + 3 = 8` |
| Ayirish | `-` | `10 - 4 = 6` |
